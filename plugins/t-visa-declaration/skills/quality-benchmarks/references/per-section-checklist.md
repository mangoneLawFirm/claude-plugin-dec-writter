# Per-Section Requirements Checklist and Phase-Level Pass/Fail Criteria

## 1. Per-Section Requirements Checklist

### Header
- [ ] Standard perjury statement present (with colon at end)
- [ ] Client full name in CAPS
- [ ] "T VISA" specified (no hyphen)
- [ ] No paragraph numbering

### Introduction (1 paragraph)
- [ ] Full name
- [ ] Date of birth
- [ ] Birthplace (city, state, country)
- [ ] Current location
- [ ] Mentions documents kept by perpetrator (if applicable)
- [ ] Purpose statement specifies trafficking type: "severe forms of labor trafficking" / "sex trafficking" / "trafficking"
- [ ] Starts paragraph numbering at 1

### Background (STRICTLY 2 paragraphs)
- [ ] STRICTLY 2 paragraphs — no 3rd paragraph without explicit attorney override
- [ ] FAIL if 3+ paragraphs
- [ ] Transport/journey details belong in Trafficking opening, NOT in Background
- [ ] Every sentence passes vulnerability test: "Does this explain WHY client became vulnerable to trafficking?"
- [ ] No general life history unconnected to vulnerability
- [ ] Children's details included if available
- [ ] Ends with clear connection to vulnerability (isolation, lack of support, emotional state)
- [ ] Immigration history included (entry dates, visa type)
- [ ] **NO INTENT TO REMAIN**: No language suggesting intent to remain permanently — FAIL if "I planned to stay," "I intended to build a life," etc.
- [ ] Entry method details: coyote = approximate date only; visa = state visa type
- [ ] Childhood vulnerability factors addressed if available (poverty, child labor, food/housing)
- [ ] **NO US VULNERABILITY PARAGRAPH** (US-met trafficker): ¶2 must NOT establish US-side desperation ("willing to take any job," "no connections," "no options," "afraid to complain") — FAIL if Background establishes pre-trafficking desperation that frames case as VAWA

### Trafficking (10-16+ paragraphs)
- [ ] **LONGEST section** by significant margin — FAIL if shorter than Physical Presence or Extreme Hardship
- [ ] **FIRST PARAGRAPH = PROCESS/SETUP**: Opens with scheme/recruitment paragraph (who, how recruited, what promised, why victim agreed, how hook was set) — NOT exploitation. FAIL if first paragraph jumps into labor/abuse.
- [ ] **AGREEMENT FRAMING**: Victim's agreement framed around trafficker's PROMISES, not victim's needs/desperation — FAIL if "I agreed because I was desperate"
- [ ] **LABOR FORMULA (¶2)**: Second paragraph describes labor with enforcement: tasks, hours, who benefited + how trafficker responded to resistance — FAIL if labor described without enforcement mechanism
- [ ] **COMMERCIAL SEX / DEBT BONDAGE**: If applicable, dedicated paragraph present — FAIL if commercial sex or debt bondage facts exist but no dedicated paragraph
- [ ] Each control method gets its OWN paragraph (no compressing)
- [ ] Follows Phase 2 paragraph plan
- [ ] EMP elements clearly covered:
  - [ ] ENDS: Outside labor + domestic labor (specific tasks, foods, chores)
  - [ ] MEANS-Force: Physical violence incidents (each separate)
  - [ ] MEANS-Fraud: False promises/deceptions (if applicable)
  - [ ] MEANS-Coercion: Threats (with exact words when available)
  - [ ] MEANS-Environment: Isolation, surveillance, document confiscation, immigration threats, economic control, movement restriction
  - [ ] PROCESS: Recruitment, harboring, obtaining — described with ACTIVE trafficking verbs ("recruited," "transported," "harbored," "exploited")
- [ ] Direct quotes used when available — ALL in English (no Spanish quotes, no Spanish parenthetical terms)
- [ ] For each control method: WHAT happened + HOW it made client feel + WHY it kept client trapped
- [ ] Escalation pattern visible across paragraphs
- [ ] Specific details preserved (foods, chores, amounts, times)
- [ ] **ACTIVE VOICE**: Trafficker is subject of action verbs — FAIL if passive constructions like "I was required," "I was taken," "I was kept"
- [ ] **THREATS IN QUOTES**: Trafficker's threats framed as direct quoted speech — flag with [ATTORNEY REVIEW] if exact words not confirmed
- [ ] **ESCAPE BARRIERS**: Isolation/confinement paragraphs explicitly explain why client could not: contact family, escape when alone, approach authorities — FAIL if reasons are only implicit
- [ ] **SEXUAL VIOLENCE LANGUAGE**: Uses trafficking terms ("sexual slave," "sexual servitude") — FAIL if uses "sexually abused," "sexual assault," or clinical framing
- [ ] **PIVOTAL EVENT CONTEXT**: Key incidents (forced entry, police involvement, separation) include contextual setup explaining WHY they occurred
- [ ] **PARAGRAPH OPENING VARIETY**: No more than 2-3 paragraphs open with the trafficker's name as the first word — FAIL if most paragraphs start with the same name
- [ ] **VAWA-vs-TRAFFICKING FILTER**: Every paragraph connects MEANS to ENDS — FAIL if any paragraph only describes emotional abuse/control without tying to labor/servitude/financial exploitation
- [ ] **MEANS-ENDS PAIRING**: Every Means paragraph explicitly ties to a specific End — FAIL if orphaned Means (threats without exploitation purpose)
- [ ] **COERCED COMPLIANCE**: Client's compliance framed as fear-driven, never voluntary — FAIL if language like "my goal was to keep the peace" or "I told her it was okay"
- [ ] **SURVEILLANCE AS TRAFFICKING TOOL**: Monitoring/surveillance framed as labor/service control, not relationship jealousy — FAIL if framed as VAWA-style controlling behavior
- [ ] **CONTROL MECHANISM ESTABLISHMENT**: Each control method explains HOW the trafficker gained that control — FAIL if control just appears without mechanism
- [ ] **ENFORCEMENT MECHANISM**: Each forced action specifies HOW compliance was enforced (threats, consequences) — FAIL if "she forced me" without explaining how
- [ ] **FINANCIAL EXPLOITATION SPECIFICS**: Financial control includes amounts, expenses, % salary, work hours — FAIL if generic "I paid for everything"
- [ ] **SERVITUDE CONTRAST**: Domestic servitude paragraphs contrast trafficker's idleness vs victim's labor — FAIL if no power dynamic contrast shown
- [ ] **RECRUITMENT EXTRACTION**: Recruitment shows how trafficker learned victim's vulnerabilities — FAIL if recruitment is just "we started a relationship"
- [ ] **THREAT IMPACT**: Threats include the specific fear/consequences the victim believed — FAIL if threats mentioned without describing their terrifying effect
- [ ] **CHILD SEPARATION**: Parent framed as wanting child but unable due to trafficking — FAIL if separation sounds voluntary
- [ ] **SCENE-GROUNDING**: At least one specific trigger scene per major control method — FAIL if all control methods described abstractly
- [ ] **CONVERSATIONAL AUTHENTICITY**: At least 3-5 natural speech markers across trafficking section (memory anchors, hedging, self-corrections)
- [ ] **WAGE EXPLOITATION FORMULA**: When pay fraud involved, includes hours worked, promised rate, expected pay, actual pay, effective rate — FAIL if flat amount stated without comparison
- [ ] **INSULT CONTENT SPECIFICITY**: Verbal abuse includes actual insults or specific nature — FAIL if only "insults," "degrading language," or "foul language" without specifics
- [ ] **FIRING FEAR PROHIBITION**: No language implying fear of being fired/losing job — HARD FAIL if "afraid they would fire me" or similar (implies voluntary employment)
- [ ] **THREAT TRANSMISSION SPECIFICITY**: Immigration threats shown as specific acts of communication to the client — FAIL if "everyone knew" or ambient atmosphere without specific scene
- [ ] **FIRST-PERSON SINGULAR ONLY**: Declaration uses "I/me" exclusively — HARD FAIL if "us/we/our" used for victim group experience (family "we" is acceptable)
- [ ] **FORWARD REFERENCE PROHIBITION**: Concepts introduced with concrete facts before being characterized as patterns — FAIL if "the constant [X]" appears before X is described
- [ ] **PLAIN LANGUAGE**: No idiomatic expressions that sound like Spanglish or literal translations — FAIL if contains idioms like "get me in the door"

### Physical Presence (1-2 paragraphs)
- [ ] **DIRECT CONNECTION STATEMENT**: Explicitly states presence is "directly connected to the trafficking" — FAIL if this causal statement is missing
- [ ] **HOW TRAFFICKING KEPT CLIENT IN U.S.**: Explains control of movement, documents, housing, fear, dependency, lack of freedom — FAIL if no causal mechanism explained
- [ ] **WHY CLIENT DIDN'T LEAVE**: Explains why client remained (fear, retaliation, lack of resources, psych harm) connected to trafficking, not personal preference — FAIL if reasons missing
- [ ] **WHAT CHANGED** (conditional): If client entered for a different reason, explains transition from original purpose to trafficking victimization
- [ ] **ESCAPE/RELEASE** (conditional): If client escaped, explains how and why they remained in U.S.
- [ ] **LEFT AND RETURNED** (conditional): If client ever left U.S. and returned, explains why (framed carefully)
- [ ] **CAUSE-AND-EFFECT LANGUAGE**: Uses explicit causal phrases: "Because of the trafficking...", "As a result of the control and fear..." — FAIL if causal language missing
- [ ] Physical/mental health consequences — ONLY symptoms from facts (if available)
- [ ] TVAP/support services status mentioned if available
- [ ] No symptoms invented or inferred
- [ ] No repetition of specific incidents already told in Trafficking
- [ ] **PP CONTENT BOUNDARY**: NO country of origin content, NO "life built here" arguments, NO immigration fears unconnected to trafficking — these belong in Extreme Hardship — FAIL if PP contains hardship content
- [ ] **PP NO RE-NARRATION**: References trafficking briefly using general language, does NOT repeat specific scenes/hours/violence from Trafficking section
- [ ] **PP LENGTH**: 1-2 paragraphs, 150-300 words — FAIL if 3+ paragraphs or >350 words

### Law Enforcement (1 paragraph)
- [ ] Correct reporting status (check supplemental notes for updates)
- [ ] Willingness to cooperate clearly stated
- [ ] If reported: agency, date, ongoing cooperation
- [ ] If not reported: readiness, what information can be provided
- [ ] **DELAYED COOPERATION EXPLAINED**: If gap between trafficking end and reporting, explains WHY (fear, trauma, danger) — FAIL if delay unexplained
- [ ] **FBI REPORTING LETTER DATE**: Specific date of reporting letter to FBI included — FAIL if missing without [ATTORNEY REVIEW] flag
- [ ] **LEA BREVITY**: 1 concise paragraph — FAIL if reads as evidence inventory
- [ ] **LEA ALTRUISTIC FRAMING**: Includes motivation for cooperating (justice, protecting others) — not just evidence listing

### Extreme Hardship (3 paragraphs; 4 max with extensive medical evidence)
- [ ] **LEGAL STANDARD CITATION**: Opens with "extreme hardship involving unusual and severe harm" + cites INA § 101(a)(15)(T)(i)(IV) and 8 C.F.R. § 214.209 — FAIL if missing
- [ ] No large [DETAILS NEEDED] placeholder blocks — argues with available facts
- [ ] Medical hardship — specific conditions, treatments, risk if interrupted
- [ ] **COUNTRY CONDITIONS — REQUIRED**: Dedicated paragraph on home country/region conditions — FAIL if missing
  - [ ] Healthcare access for client's specific conditions in home region
  - [ ] Disability infrastructure/accessibility (if applicable)
  - [ ] Safety conditions (general + specific to client)
  - [ ] Economic conditions and ability to self-support
  - [ ] If specific data unavailable, general argument with [ATTORNEY REVIEW] flag for evidence
- [ ] Family separation impact — dependents, loss of support system, caregiver dependency
- [ ] Safety fears — personal threats from trafficker connected to home country
- [ ] Legal process disruption (ongoing cooperation, case participation)
- [ ] **EH BREVITY**: 3 paragraphs max (4 only with extensive medical evidence) — FAIL if 5+ paragraphs
- [ ] **EH WORD COUNT**: ~250-400 words — FAIL if exceeds 500 words (indicates re-narration)
- [ ] **EH FORWARD-LOOKING**: Arguments focus on removal consequences, not re-narration of trafficking — FAIL if re-elaborates
- [ ] Safety fears and country conditions merged into ONE paragraph

### Moral Character (2-3 paragraphs)
- [ ] Immigration violations addressed honestly with genuine remorse ("I sincerely apologize")
- [ ] Waiver request connected to law enforcement cooperation
- [ ] Non-violence emphasized (restraint despite mistreatment)
- [ ] Work ethic and honest labor
- [ ] Character traits: compassion, humility, community contribution
- [ ] Tax info: omit if missing (do NOT flag with placeholder)

### Conclusion (1 paragraph)
- [ ] Tone: "I respectfully and humbly ask" / "I ask you" / "I beg you" — emotionally direct
- [ ] Brief reaffirmation of victim status (one clause, not a recap)
- [ ] Request for protection under T Nonimmigrant Status
- [ ] Statement that information is truthful
- [ ] Desire to heal + commitment to law + gratitude
- [ ] Feels like a heartfelt plea, not a formal closing
- [ ] **CONCLUSION BREVITY**: 3-5 sentences maximum — FAIL if recaps specific trafficking details, dates, or evidence already presented

## 3. Phase-Level Pass/Fail Criteria

### Phase 1: Extraction
| Criterion | Pass | Fail |
|-----------|------|------|
| Valid JSON output | Parses without errors | Malformed JSON |
| Schema compliance | All fields present (empty string for missing) | Missing schema fields |
| Granularity | Specific foods, chores, tasks, quotes captured | Generic categories only ("cooked," "cleaned") |
| Direct quotes | Preserved in original language + English | Quotes lost or paraphrased |
| Empty fields | Empty when info not in transcript | Fields filled with inferred data |
| Completeness | >95% of transcript facts captured | Major facts omitted |

### Phase 2: Classification
| Criterion | Pass | Fail |
|-----------|------|------|
| Valid JSON output | Parses without errors | Malformed JSON |
| Background filter | Only vulnerability-connected facts assigned | General life history included |
| Background weight | MAX 2 paragraphs assigned | 3+ paragraphs worth of facts |
| Trafficking paragraph plan | 10-16+ planned, first = Process/Setup, second = Labor+Enforcement | <10 planned, missing Process/Setup first paragraph, multiple methods compressed |
| EMP mapping | All 3 elements mapped with detail | Missing elements or generic mapping |
| Gap analysis | Every empty field flagged with impact + follow-up question | Gaps missed or generic flags |
| Strategic recommendations | Present for weak sections | Missing for sections with thin evidence |

### Phase 3: Section Writers
| Criterion | Pass | Fail |
|-----------|------|------|
| Paragraph limits | Background STRICTLY 2, Trafficking 10+, PP 1-2, EH 3 (4 max), others per spec | Background 3+, Trafficking <10, PP 3+, EH 5+, EH missing country conditions |
| Fact grounding | Every claim traceable to provided facts | Invented or inferred details |
| Prohibited words | 0 violations (incl. sexually abused/assault) | Any violation |
| English-only | Entire declaration in English, all quotes translated | Any Spanish words, phrases, or untranslated quotes |
| Reading level | 6th-8th grade | Academic/legal language |

| Criterion | Pass | Fail |

| Prohibited words | 0 remaining violations (incl. sexually abused/assault) | Any violation |

### Phase 5: Notes
| Criterion | Pass | Fail |
|-----------|------|------|
| Prioritization | Gaps coded by severity (HIGH/MEDIUM/LOW) | Flat list without priority |
| Actionability | Specific follow-up questions provided | Generic "more info needed" |
| Completeness | All gaps from Phase 2 addressed | Gaps missed |
| Source distinction | Differentiates client follow-up vs attorney/supplemental | All gaps treated the same |
