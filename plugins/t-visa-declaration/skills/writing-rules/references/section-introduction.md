# Introduction Section Instructions

## Format Rules
- **Paragraph count**: 1 paragraph only
- **Numbering**: Start at paragraph 1
- **Heading**: NO heading — no "I. INTRODUCTION" or any variant. HARD FAIL if heading appears.

## Required Elements (ALL must be present)
1. **Full legal name** — as it appears in all documents
2. **Date of birth** — complete format (month/day/year)
3. **Birthplace** — city, state/province, country
4. **Current location** — where client currently resides (city/state or city/country)
5. **Documents kept by perpetrator** — if applicable to trafficking situation
6. **Purpose statement** — specify which trafficking type applies:
   - Sexual trafficking
   - Labor trafficking
   - Mixed (labor + sexual)

## Content Approach
- State facts plainly and directly
- No narrative flourish
- Introduce self and establish foundational identity information
- Set stage for trafficking narrative that follows
- Make clear what type of trafficking client experienced

## CRITICAL RULE: This paragraph establishes who is making the declaration and provides essential identifying information. Every element listed above must appear naturally within this single paragraph.
